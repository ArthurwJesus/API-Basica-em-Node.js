# API basica desenvolvida com Node.JS :rocket:

inicie um projeto com os seguintes passos e logo apos você pode usar os outros arquivos:
<li>npm init</li>
<li>npm install --save express</li>
<li>Exclui a pasta Node.modules e faça o comando npm install para instalar novamente a pasta</li><br/>

Para executar a Api abra o terminal e faça o seguinte comando:
<li> node src/index.js </li><br/>
